# 数据库设计

本项目提交版本采用 SQLite 数据库，数据库文件位于：

```text
backend/data/fintel.db
```

## 1. news_raw 新闻原始表

| 字段 | 类型 | 说明 |
|---|---|---|
| id | INTEGER | 新闻 ID |
| title | TEXT | 标题 |
| content | TEXT | 正文 |
| source | TEXT | 来源 |
| publish_time | TEXT | 发布时间 |
| url | TEXT | 新闻链接 |
| stock_code | TEXT | 股票代码 |
| stock_name | TEXT | 股票名称 |
| industry | TEXT | 所属行业 |

## 2. events 事件表

| 字段 | 类型 | 说明 |
|---|---|---|
| event_id | INTEGER | 事件 ID |
| event_title | TEXT | 事件标题 |
| event_summary | TEXT | 事件摘要 |
| first_source | TEXT | 首发来源 |
| first_publish_time | TEXT | 首次发布时间 |
| event_type | TEXT | 事件类型 |
| duplicate_count | INTEGER | 重复新闻数量 |

## 3. analysis_results 分析结果表

| 字段 | 类型 | 说明 |
|---|---|---|
| analysis_id | INTEGER | 分析 ID |
| event_id | INTEGER | 事件 ID |
| sentiment | TEXT | 情绪方向 |
| sentiment_score | REAL | 情绪分 |
| impact_score | REAL | 影响分 |
| risk_score | REAL | 风险分 |
| confidence_score | REAL | 置信度 |
| related_stock_code | TEXT | 相关股票代码 |
| related_stock_name | TEXT | 相关股票名称 |
| related_industry | TEXT | 相关行业 |
| analysis_reason | TEXT | 分析理由 |
| summary | TEXT | 分析摘要 |
| event_type | TEXT | 事件类型 |

## 4. kg_nodes 图谱节点表

| 字段 | 类型 | 说明 |
|---|---|---|
| node_id | TEXT | 节点 ID |
| name | TEXT | 节点名称 |
| node_type | TEXT | 节点类型 |
| properties | TEXT | JSON 属性 |

## 5. kg_edges 图谱关系表

| 字段 | 类型 | 说明 |
|---|---|---|
| id | INTEGER | 关系 ID |
| source_id | TEXT | 起点 |
| target_id | TEXT | 终点 |
| relation_type | TEXT | 关系类型 |
| weight | REAL | 关系权重 |
| properties | TEXT | JSON 属性 |
