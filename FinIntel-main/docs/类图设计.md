# 类图设计说明

核心类包括：

- NewsItem：新闻对象
- EventItem：事件对象
- AnalysisResult：分析结果对象
- GraphNode：知识图谱节点
- GraphEdge：知识图谱关系
- BaseCrawler：抽象爬虫类
- MockNewsCrawler：模拟新闻采集类
- CrawlerManager：多源采集调度类
- NewsCleaner：新闻清洗类
- DeduplicationService：去重服务类
- LLMAnalyzer：智能分析类
- DatabaseManager：数据库管理类
- NewsRepository：新闻数据访问类
- EventRepository：事件数据访问类
- AnalysisRepository：分析结果访问类
- GraphRepository：图谱数据访问类
- KnowledgeGraphService：知识图谱构建类
- IntelligencePipeline：总流程调度类

后续可以根据这些类绘制 UML 类图。
