# 接口说明文档

## 1. 运行情报分析流水线

```text
POST /api/pipeline/run
```

参数：

```text
clear_before_run: 是否清空数据库后重新运行
```

## 2. 获取新闻列表

```text
GET /api/news
```

## 3. 获取事件列表

```text
GET /api/events
```

## 4. 获取分析结果

```text
GET /api/analysis
```

## 5. 获取统计数据

```text
GET /api/statistics
```

返回内容包括：

- 新闻总数
- 事件总数
- 情绪分布
- 新闻来源分布
- 行业影响分布
- 影响分 Top10 事件

## 6. 获取知识图谱数据

```text
GET /api/graph
```

返回内容：

```json
{
  "nodes": [],
  "edges": []
}
```
